## 다음 단계: 모델 학습 실행

### 방법 1: 빠른 테스트 (현재 설정)

```bash
python3 03_train_classification_model.py
```

- 에폭 수: 5 (빠른 테스트용)
- 모델: EfficientNet-B3
- 배치 사이즈: 8

### 방법 2: 실제 학습 (권장 설정)

`03_train_classification_model.py` 파일의 config 부분을 다음과 같이 수정:

```python
config = {
    'batch_size': 16,  # GPU가 있다면 더 크게
    'num_epochs': 20,  # 실제 학습용
    'learning_rate': 1e-4,
    'num_workers': 4,
    'image_size': 512,
    'model_name': 'efficientnet_b4',  # 더 큰 모델
    'save_dir': 'checkpoints',
    'random_state': 42
}
```
